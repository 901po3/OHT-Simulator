1. Get Vercel API Token:
   Visit: https://vercel.com/account/tokens
   Create new token, copy it

2. Get Project IDs (after initial deployment):
   Dashboard: https://vercel.com/dashboard
   Note: VERCEL_ORG_ID and VERCEL_PROJECT_ID

3. Add GitHub Secrets:
   Repository: https://github.com/901po3/OHT-Simulator
   Go to: Settings → Secrets and variables → Actions
   
   Add 3 secrets:
   - VERCEL_TOKEN: (paste API token)
   - VERCEL_ORG_ID: (paste org ID)
   - VERCEL_PROJECT_ID: (paste project ID)

4. Push changes to GitHub:
   git push origin master
   
5. Automatic deployment starts:
   Go to: Actions tab in GitHub
   Watch workflow execute automatically
