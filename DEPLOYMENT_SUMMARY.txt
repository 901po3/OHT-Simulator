7. DEPLOYMENT OPTIONS
================================================================================

OPTION A: Manual Deployment (Immediate)
────────────────────────────────────────
1. Open PowerShell, navigate to project:
   cd C:\Unity\Portfolio\OHT-Simulator

2. Login to Vercel:
   vercel login
   (Follow browser prompt)

3. Link project (first time only):
   vercel link
   (Follow prompts)

4. Deploy to production:
   vercel --prod

5. Access at provided URL (e.g., https://oht-simulator.vercel.app)

Time required: 5-10 minutes
Recommended for: First deployment, quick testing

OPTION B: Automatic via GitHub Actions (Recommended for ongoing)
─────────────────────────────────────────────────────────────
1. Get Vercel API token:
   https://vercel.com/account/tokens

2. Add GitHub secrets:
   https://github.com/901po3/OHT-Simulator/settings/secrets/actions

   Add:
   - VERCEL_TOKEN=<your_token>
   - VERCEL_ORG_ID=<your_org_id>
   - VERCEL_PROJECT_ID=<your_project_id>

3. Push changes:
   git push origin master

4. Automatic deployment:
   - GitHub Actions runs workflow
   - Vercel deployment starts automatically
   - Check Actions tab for progress

Time required: <2 minutes setup
Recommended for: Ongoing production deployment
